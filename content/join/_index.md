    #     2026 年，鲁东大学计算机与人工智能学院智能信息处理实验室拟招收 **081200 计算机科学与技术学术学位硕士研究生**与 **085410 人工智能专业学位硕士研究生**。导师信息见[研究团队](/people/)。

    #     实验室招收的第一批 4 名研究生将于 2026 年 6 月毕业，已分别被南开大学、同济大学、北京工业大学和江南大学录取攻读博士研究生。

    #     计算机科学与技术学硕调剂缺额 25+，人工智能专硕调剂缺额 17+。

    #     根据学院公告，调剂要求如下：

    #     - 初试成绩符合第一志愿报考专业和调入专业 A 类考生（一区）国家线；
    #     - 计算机科学与技术学硕要求第一志愿报考专业为计算机科学与技术、信息与通信工程、控制科学与工程、软件工程、网络空间安全、电子信息或智能科学与技术，且考试科目为英语（一）、数学（一）；
    #     - 人工智能专硕要求第一志愿报考专业为电子信息（0854），且考试科目为英语（一）或英语（二），业务课一为数学（一）或数学（二）。

    #     由于本年度首次调整为 408 科目考试，暂无历年调剂分数线数据可供参考，建议学硕总分 280+、专硕总分 300+。

    #     欢迎符合以上要求、对食品计算与 Agent、具身智能相关研究方向感兴趣的同学联系调剂。

    #     有意向的同学请填写[2026 年鲁东大学智能信息处理实验室调剂申请表](https://docs.qq.com/form/page/DUUdUTWRoZmdyZmdL)，并添加微信 `15753724366`，注明个人信息。
